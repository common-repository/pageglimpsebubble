PageGlimpse Bubble integrates PageGlimpse web page previews in your Wordpress blog.

== Description ==

The PageGlimpse Bubble is a simple script used to display a bubble showing a thumbnail of the webpage for a link in your blog, using the PageGlimpse service. Works with all browsers and is very easy to install and administer.

Learn more at http://blog.pageglimpse.com

The javascript code includes code from xEvent, Copyright 2001-2006 Michael Foster (Cross-Browser.com), part of X, a Cross-Browser Javascript Library, Distributed under the terms of the GNU LGPL.


== Installation ==

1. Upload the folder PageGlimpseBubble to the Wordpress plugin directory "/wp-content/plugins/" 
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the plugin options page and you can:
	- enable/disable the bubble
	- enter your PageGlimpse developer key which you need in order to use the service
